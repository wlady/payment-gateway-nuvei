WooCommerce Nuvei Gateway.

== Description ==

Allow users to make payments with credit / debit cards via Nuvei.

== Frequently Asked Questions ==

= Is this PCI DSS compatible? =

Yes, absolutely. Visit Nuvei website for more info.

= Can I use this gateway for tests? =

Yes, you can use Nuvei's sandbox account and test credit cards numbers.
